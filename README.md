# How to build:
1. ### Install CLIs:
* ##### Prerequisites
> https://tauri.app/v1/guides/getting-started/prerequisites
* ##### Install cli
> ##### cargo install tauri-cli

2. ### Build:
> ##### cd HyperGear

> ##### npm install

> ##### cd..

> ##### cargo tauri build

# How to debug
> ##### cargo tauri dev
> ##### main window ctrl+shift+i
